# ICP - AI Komponentti

React-sovellus AI komponentin kehittämiseen.

## Teknologiat

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Kehitys

Asenna riippuvuudet:

```bash
npm install
```

Käynnistä kehityspalvelin:

```bash
npm run dev
```

## Rakenne

- `src/App.tsx` - Pääkomponentti AI chat -toiminnallisuudella
- `src/index.css` - Tailwind CSS tyylit
- `tailwind.config.js` - Tailwind konfiguraatio
